


注意:

需在设置里 `Groups` 填入需启用的 QQ群号



## 自定义问答

> 1. 使用 `AdminQQ` (或者在本群内是群主/管理员) 在群内 发送以下格式消息

```
#问 
自定义关键词   
#答 
自定义回答
```

> 2. 在群内 发送 `#已学习` 获取已学习问答

> 3. 在群内 发送 `#删除问答 关键词` , 删除指定问答


## 在群聊里使用

> 目前群需在 `Groups` 内

> 当群内消息包含自定义关键词时, 机器人就会自动回复自定义回答
